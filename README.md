## Customization

### Add Your Images
Replace image URLs in HTML files:
```html
<img src="your-image-url.jpg" alt="Description" loading="lazy">
```

### Change Colors
Edit color variables in `css/styles.css`:
```css
:root {
    --primary: #ff6b6b;
    --secondary: #1a1a2e;
    --dark: #16213e;
}
```

### Modify Text Content
Edit directly in HTML files. All text is clear and easy to find.

### Add/Remove Pages
1. Create new HTML file with similar structure
2. Add navigation link in navbar
3. Create corresponding JS file if needed
